# Быстрый переход между комнатами хаба

Добавлена отдельная группа кнопок в верхнюю панель интерфейса:

- Market
- Arena
- Archive

## Поведение

- Кнопки отображаются только внутри раздела Hub.
- Активная комната подсвечивается своим цветом.
- Кнопка `Hub` из основной навигации возвращает в центральную комнату.
- Переход выполняется с существующей анимацией затемнения сцены.
- Адрес синхронизируется с комнатой: `#/?room=market`, `#/?room=arena`, `#/?room=archive`.
- Прямое открытие такого адреса сразу загружает нужную комнату.
- Переходы через двери внутри хаба также обновляют адрес.
- Старое дублирующее меню внутри сцены удалено.
